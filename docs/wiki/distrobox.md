# distrobox

**distrobox** is a tool that allows you to use any Linux distribution inside
your terminal. It uses Docker or Podman to create highly integrated containers
that share your home directory, external storage, external USB devices, and GUI
apps.

## Installation

Once the ScientiOS repository is configured, install `distrobox` via:

```bash
sudo xbps-install -Sy distrobox
```
